# Runtimes

## Dependency compatibility

Dependencies of incompatible runtime are forbidden.

- The runtime `http://www.scala-lang.org/2021/scala/3/generic` is compatible with all other runtimes.
- The runtime `http://www.scala-js.org/2013/scala/3` is compatible with:
  - `http://www.scala-js.org/2013/scala/3/web`
  - `http://www.scala-js.org/2013/scala/3/node`
